# synet


* This project aimed to create CNN + LSTM models for 6 ranscription factors


# Requirements
    * keras==2.8.0
    * keras-preprocessing==1.1.2
    * tensorflow==2.8.0
    * scipy=1.7.3
    * numpy=1.21.2


# setup environment
* The conda env can be created using **synet.yml** file
